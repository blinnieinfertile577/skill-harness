## 🚀 What skill-harness does

skill-harness is an installer and agent harness for the skill-pack suite. It helps you set up and use skill packs across Claude and Codex with less manual work.

Use it when you want a single place to:

- install skill packs
- manage agent tools
- keep Claude and Codex setups aligned
- run helper agents for day-to-day work
- reduce repeat setup steps

This app is built for end users who want the software to work without a long setup process.

## 💻 Before you start

Use a Windows PC with:

- Windows 10 or Windows 11
- an internet connection
- at least 2 GB of free disk space
- permission to install or run apps
- a modern browser such as Edge, Chrome, or Firefox

For best results, close other large apps before you start the install.

## 🛠️ How to use skill-harness

skill-harness acts as a setup hub for the skill-pack suite. In normal use, you will:

1. Open skill-harness
2. Choose the tool you want to set up
3. Select the skill pack or agent bundle you want
4. Let the app install or link the files
5. Open Claude or Codex and use the new setup

Typical tasks include:

- adding new skills
- updating existing skill packs
- setting up helper agents
- switching between Claude and Codex profiles
- checking what is already installed

If you work with both Claude and Codex, this app can help keep the setup process in one place.
